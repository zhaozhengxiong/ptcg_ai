# PTCG Admin Console

Next.js 14 admin console for monitoring matches, manual confirmation, and case management.

## Setup

```bash
cd apps/admin_console/frontend
npm install
npm run dev
```

## Features

- Match monitoring dashboard
- Manual confirmation interface for Referee decisions
- Case library browser for rule precedents
- Configuration for auto/manual mode thresholds
- Auth0 integration (to be configured)
